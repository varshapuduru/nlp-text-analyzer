# nlp-text-analyzer
# 🧠 NLP Text Analyzer

A powerful NLP-based web application built using **Python, spaCy, and Streamlit** that analyzes text and provides linguistic insights.

---

## 🚀 Features

- 🔤 Tokenization
- 🧩 Part-of-Speech (POS) Tagging
- 🔍 Lemmatization
- 🏷️ Named Entity Recognition (NER)
- 😊 Sentiment Analysis
- 🎨 Stylish UI with background and glass design
- ⚡ Smart fallback for entity detection

---

## 🛠️ Tech Stack

- Python 🐍
- Streamlit 🌐
- spaCy 🧠
- TextBlob 😊
- Pandas 📊

## ⚙️ Installation & Run

```bash
# Clone the repo
git clone https://github.com/your-username/nlp-text-analyzer.git

# Navigate to project
cd nlp-text-analyzer

# Install dependencies
pip install -r requirements.txt

# Download spaCy model
python -m spacy download en_core_web_sm

# Run app
streamlit run app.py
